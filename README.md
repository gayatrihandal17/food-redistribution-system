# Food Waste Redistribution Platform

A full-stack web application built using Spring Boot and MySQL to reduce food wastage by connecting Hotels with NGOs.

## Features
- Hotels can add surplus food with expiry time
- NGOs can view available food and request pickup
- Admin dashboard to monitor food listings and requests
- Role-based pages (Hotel, NGO, Admin)

## Technologies Used
- Java
- Spring Boot
- MySQL
- HTML, CSS, JavaScript

## How to Run
1. Clone the repository
2. Import into Eclipse as Maven project
3. Configure MySQL in application.properties
4. Run FoodredistributionApplication.java
5. Open browser: http://localhost:8080/login.html

## Project Objective
To minimize food waste and support NGOs by redistributing surplus food efficiently.
